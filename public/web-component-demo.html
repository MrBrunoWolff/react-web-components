Web Components Demo

Button Web Component

This demonstrates our UI Button as a web component that can be used in any web application, not just React.

Usage Example:

<ui-button variant="default" size="default">Default Button</ui-button>

Variants:

Default Destructive Outline Secondary Ghost Link

Sizes:

Default Size Small Large

Disabled State:

Disabled Button

Event Handling

You can attach events to web components just like regular HTML elements:

const button = document.querySelector('ui-button#clickMe'); button.addEventListener('click', () => { alert('Button clicked!'); });
